# LabTask10

**From arrays to action: Exploring the power of 2D matrix operations.**

## Description
This project demonstrates the use of **2D arrays** in Java. It includes programs for common matrix operations such as:

- Matrix Addition  
- Matrix Multiplication  
- Matrix Transpose  
- Row and Column Summation  
- Diagonal Sum  

Each program is implemented in a separate Java class under the `src` folder.
